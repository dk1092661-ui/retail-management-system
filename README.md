# Retail Management System

A modern Retail Management System built with **Python**, **CustomTkinter** (beautiful desktop UI), **SQLite**, and **Matplotlib**.  
Features: Inventory CRUD, Sales / Billing, Reports (weekly/monthly), and a clean sidebar navigation.

---

## Demo & Screenshots

> Open the repo to see the app files and screenshots (add screenshots to `/assets` and link them here).

---

## Key Features

- Modern, dark-themed desktop UI using CustomTkinter  
- Inventory management: add / edit / delete products  
- Sales management: record sales, auto-update stock  
- Reports: Today / Weekly / Monthly sales charts embedded in-app  
- Single-window sidebar navigation (no multiple popups)  
- Small, polished utilities: export-ready design, CV included

---

## Tech Stack

- Python 3.10+  
- CustomTkinter (UI)  
- SQLite (local database)  
- Matplotlib (charts)  
- Pillow (images, icons)
